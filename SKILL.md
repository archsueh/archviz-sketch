# archviz-sketch

> Sketch illustration pipeline — article illustrations with hand-drawn feel.

## When to Use

- Article illustrations that need a "process sketch" aesthetic
- Concept diagrams with construction lines and imperfect hand-feel
- Teaching illustrations that look drawn, not generated
- CJK-optimized illustration layouts

## When NOT to Use

- Data charts, flowcharts, architecture diagrams → **archviz**
- 3D building/structure visualization → **archviz-3d**
- Photorealistic images → imagegen / fal MCP

## Skill Boundaries

| Need | Use |
|---|---|
| 2D diagram/chart | archviz |
| 3D visualization | archviz-3d |
| Sketch/illustration | **this skill** |
