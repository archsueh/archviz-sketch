# 极简线条 Prompt Template (for archviz-sketch)

**Trigger:** 极简线条, minimal line, 线稿, 概念视觉

**Base Prompt:**

Create a high-resolution 2K image of "{{subject}}" in clean minimal line illustration style.

Visual language: Fine thin black lines, selective vibrant color accents (max 2-3 colors), strong negative space, elegant composition.

Composition: {主视觉 + 辅助元素 + 少量标注}. Layers: main subject + supporting lines + integrated title if needed.

Title: "{标题}" subtly integrated into the line work if appropriate.

Core concept: {视觉隐喻或关键关系}

Constraints:
- Extremely restrained: no fills, minimal shading (light hatching only if essential)
- Precise but hand-drawn feel (subtle wobble on lines)
- High contrast line weight variation for hierarchy
- Generous white space (at least 40%)
- No decorative elements, no gradients, no textures beyond the line itself

Negative prompt:
Thick lines, heavy shading, photorealism, 3D rendering, excessive decoration, cluttered composition, low contrast.

**Quality Checklist:**
- Lines are thin and elegant
- Negative space feels intentional and breathable
- Color used only for emphasis on key relationships
- Readable at multiple scales

**Usage in Pipeline:**
Ideal for covers, posters, concept visuals where clarity and sophistication matter. Combine with process-gao when showing evolution.

**Integration with 400-free:**
Reference "Lukasz Adam" or "Open Doodles" line quality for the subtle hand feel while staying minimal.
